How To Include CSS

  • Inline CSS Style

    1. Not a good approach

      1. Adding CSS rules to each HTML element takes time and makes your HTML structure unorganized. It's difficult to keep up, reuse, and scale.

    2. Can be used

      1. Inline takes precedence over all other styles. Any styles defined in the internal and external style sheets are overridden by inline styles.

  • The <Style> Element

    1. Not a good approach

      1. Difficult in styling multiple html document.

    2. Can be used

      1. Style same type of elements.

  • The External Style Sheet

    1. Recommended

      1. More Flexible
